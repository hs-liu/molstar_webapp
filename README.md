Mol* Webapp
===========

Develop using Typescript via React with Vite framework 
Implemented with Mol* Package
Using HTTP queries to retrieve data from server: see ```molstar_webapp_server``` repo 

To Run on Local Machine:
------------
run on terminal: 
```
npm install
npm run dev
```
